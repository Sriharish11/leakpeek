# LeakPeek

All-In-One Web Vulnerability CLI Scanner

## Features
- Scan for common web vulnerabilities (XSS, Open Redirect, CSRF, etc.)
- Beautiful CLI output with colors, tables, and progress bars
- Save results to CSV or JSON
- Modular and extensible

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python3 leakpeek.py scan --url https://example.com --modules all --threads 10 --output my_report.csv
```
